## Características de esta versión

- Personalización de funciones y nombres para mayor claridad y consistencia.
- Integración de nuevas configuraciones y utilidades desarrolladas por Heriberto Pat Moo.
- Ejemplos listos para enviar mensajes de texto, multimedia, encuestas, contactos y ubicación.
- Administración completa de grupos: agregar, eliminar, promover/demote, cambiar nombre y descripción.
- Soporte para mensajes efímeros (disappearing messages) y view-once.
- Funciones de privacidad y presencia: bloquear/desbloquear, último visto, en línea, escribiendo.
- Compatibilidad con broadcast y stories.
- Mejor manejo de media: descarga, re-subida y envío eficiente con streams o URLs.
